# Rosy Byte

A dark VS Code theme with hot pink accents.

## Install

Search **Rosy Byte** in the VS Code Extensions panel, or install via the command palette:

```
ext install AndreeaIsabela.rosy-byte
```

## Color Palette

| Role | Color |
|---|---|
| Background | `#1A1A1A` |
| Deep pink | `#FF1493` |
| Hot pink | `#FF69B4` |
| Blush pink | `#E8A0BF` |
| Gold (strings) | `#FFD700` |
| Mint (numbers) | `#00FF88` |
| Cyan (types) | `#00B7EB` |
